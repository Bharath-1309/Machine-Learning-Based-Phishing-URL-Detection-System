# Machine Learning Based Phishing URL Detection System

## Overview

This project is a machine learning based web application that predicts whether a given URL is phishing or legitimate.

The system uses a Decision Tree classifier trained on a phishing URL dataset. Users can enter a URL through a web interface, and the Flask backend returns the prediction.

---

## Features

- Detects phishing URLs
- Machine Learning based prediction
- Flask REST API
- Simple web interface
- Fast prediction


